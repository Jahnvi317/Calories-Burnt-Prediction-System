# Calories-Burnt-Prediction-System
This project is a Machine Learning-powered web application that predicts the number of calories burnt during physical activity based on user-specific data.It leverages a Random Forest Regressor model and provides an interactive interface built with Streamlit.

📊 Project Overview
Predicting calorie expenditure is vital for fitness tracking and health management. This project uses a dataset containing over 15,000 entries of exercise data, including physical attributes and workout intensity metrics.

Key Features:
> Exploratory Data Analysis (EDA) including Correlation Heatmaps and Scatter Plots.

> Achieved a Mean Absolute Error (MAE) of 1.68, ensuring highly precise predictions.

>  A user-friendly web app where users can input their data via sliders and get instant results.

🛠️ Tech Stack
Language: Python

Libraries: Pandas, NumPy, Scikit-Learn, Matplotlib, Seaborn

Machine Learning: Random Forest Regressor

Deployment: Streamlit Cloud / Local Host

📈 Model Performance
Through feature engineering and correlation analysis, it was determined that Duration and Heart Rate are the most significant predictors of calorie burn. The model was validated using an 80-20 train-test split.
